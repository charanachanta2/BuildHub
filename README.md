# 🚀 BuildHub

BuildHub is a full-stack web platform inspired by LeetCode, designed for
**project-based learning, team collaboration, and peer review**.

It allows users to work on real-world projects, collaborate in teams,
submit solutions, and get evaluated by reviewers.

------------------------------------------------------------------------

## 🌟 Key Features

### 👤 User System

-   User registration & login\
-   Unique User ID generation (BHxxxx)\
-   Profile with GitHub and bio

------------------------------------------------------------------------

### 👥 Team Collaboration

-   Create teams (up to 6 members)\
-   Add members using User IDs\
-   Team-based project participation

------------------------------------------------------------------------

### 📦 Project System

-   Projects uploaded using Markdown\
-   Clean project detail view\
-   Structured problem statements

------------------------------------------------------------------------

### 🚀 Submission System

-   One submission per team\
-   Submit GitHub repository and live website

------------------------------------------------------------------------

### 👨‍⚖️ Review System

-   Reviewer login system\
-   Random reviewer assignment\
-   Open review pool (any reviewer can take reviews)\
-   No self-review for fairness

------------------------------------------------------------------------

### 🔔 Notifications

-   Team member notifications\
-   Real-time updates

------------------------------------------------------------------------

### 📊 Scoring System

-   Score (0--100) and feedback\
-   Average score displayed on project page

------------------------------------------------------------------------

## 🛠 Tech Stack

-   Frontend: HTML, CSS, JavaScript\
-   Backend: Flask (Python)\
-   Database: MongoDB Atlas\
-   Markdown rendering

------------------------------------------------------------------------

## ⚙️ Installation

``` bash
git clone https://github.com/your-username/buildhub.git
cd buildhub
pip install -r requirements.txt
python app.py
```
