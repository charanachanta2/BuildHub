# 🚀 AI Resume Analyzer & Scorer

## 🎯 Objective
Build a web application that analyzes resumes and provides feedback using AI or rule-based logic.

---

## 🧠 Features

### 📄 Resume Upload
- Upload resume in PDF format
- Extract text from PDF

---

### 🧠 Resume Analysis
Analyze the resume based on:
- Skills mentioned
- Keywords (AI, Python, Web, etc.)
- Experience section
- Education

---

### 📊 Scoring System
Provide a score out of 100 based on:
- Completeness
- Skills relevance
- Formatting

---

### 💡 Suggestions
Give feedback like:
- Add more projects
- Improve skills section
- Add measurable achievements

---

### 📈 Dashboard
- Show past uploaded resumes
- Track score improvements

---

## 🛠 Tech Stack

### Frontend
- HTML, CSS
- JavaScript

### Backend
- Flask (Python)

### Database
- MongoDB

### Optional AI Tools
- OpenAI API
- NLP libraries (spaCy / NLTK)

---

## ⭐ Bonus Features

- Compare resume with job description
- ATS (Applicant Tracking System) score
- Highlight missing keywords
